HTML + JavaScript

HTML ja JavaScript ovat eri kieliä (yleensä JavaScript sivuston loppuun).

Jos haluat muokata nettisivua, esim. tehdä lomakkeen, sinun täytyy antaa id HTML tägiin/tägeihin

JavaScriptissä komennolla var elementti = document.getElementById("jokuid") saat elementin käyttöön JavaScript koodissa.

esim. var lomakkeenArvo = elementti.value (lomakke-kentän arvo)

esim.2 elementti.innerHTML = "Joku teksti sivulle" (päivittää tekstin)

document.createElement(...) ja element.appendChild(...) komennoilla saa tehtyä monimutkaisempia muutoksia.

Lisäksi täytyy vain selvitellä JavaScriptin ja HTML koodin yhteistoimintaa...
